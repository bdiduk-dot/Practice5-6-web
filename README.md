# Практична робота №2: Основи CSS

Багатосторінковий HTML-сайт з CSS-оформленням, адаптивною версткою та інтерактивною галереєю.

## 🚀 Запуск

Відкрийте `index.html` у браузері або використайте Live Server у VS Code.

## 📁 Структура

```
├── index.html           # Головна сторінка
├── pages/
│   ├── about.html      # Про мене
│   └── contact.html    # Контакти
├── styles/
│   ├── style.css       # Основні стилі
│   └── responsive.css  # Адаптивність
├── scripts/
│   └── gallery.js      # Інтерактивна галерея
└── assets/             # Зображення та медіа
```

## ✨ Що реалізовано

- Адаптивний дизайн (768px, 480px, 1200px)
- Flexbox верстка (навігація, галерея, двоколонкові блоки)
- Інтерактивна галерея з модальним вікном
- Стилізовані форми з валідацією
- Таблиці з hover-ефектами
- SVG іконки в контактах
- CSS змінні для кольорів
- Плавні переходи та анімації

## 🎨 Технології

HTML5 • CSS3 • JavaScript • Flexbox • Media Queries
