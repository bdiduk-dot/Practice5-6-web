# Практична робота №7-8: Додавання JavaScript до HTML/CSS-мінісайту

Багатосторінковий мінісайт, розширений клієнтською логікою на JavaScript: робота з DOM,
подіями, станами інтерфейсу, localStorage, валідацією форми та динамічним рендером контенту.

## Запуск

1. Відкрийте проєкт у VS Code.
2. Запустіть локальний сервер (наприклад, Live Server).
3. Відкрийте сторінку index.html через http://localhost...

Примітка: для завантаження data/services.json потрібен саме локальний сервер, а не file://.

## Структура

```
index.html
pages/
	about.html
	contact.html
styles/
	style.css
	responsive.css
js/
	main.js
	ui.js
	contact.js
data/
	services.json
assets/
	img/
```

## Реалізовані обов'язкові пункти

1. Підключення зовнішніх JS-файлів через defer на сторінках сайту.
2. Підсвічування активної сторінки в навігації (клас is-active).
3. Мобільне меню з кнопкою-бургером, aria-expanded та автозакриттям.
4. Перемикач світлої/темної теми із збереженням у localStorage (siteTheme).
5. Кнопка Повернутися вгору та динамічний рік у footer.
6. FAQ-акордеон з керуванням hidden/aria-expanded.
7. Фільтрація карток послуг за категоріями.
8. Модальне вікно/lightbox для перегляду зображень.
9. Покращена форма контактів: валідація, повідомлення помилок, лічильник символів.
10. Чернетка форми в localStorage (contactDraft), submit через FormData і блок підтвердження.

## Додатково (завдання з зірочкою)

- Рендер карток з локального JSON через fetch().
- Стани завантаження/помилки/порожніх даних.
- Обране (favoritesList) з localStorage.

## Технології

HTML5, CSS3, JavaScript (ES6+), localStorage, FormData, Fetch API.
