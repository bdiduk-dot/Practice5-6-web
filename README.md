# Практична робота №2: Основи CSS

## Опис проєкту

Це багатосторінковий HTML-сайт з повним CSS-оформленням, створений для демонстрації навичок веб-розробки. Проєкт включає три сторінки з єдиним дизайном, адаптивною версткою та сучасним інтерфейсом.

## Структура проєкту

```
lab02-css/
├─ index.html              # Головна сторінка
├─ pages/
│  ├─ about.html          # Сторінка "Про мене"
│  └─ contact.html        # Сторінка контактів
├─ assets/
│  ├─ img/                # Зображення
│  └─ media/              # Відео та аудіо файли
├─ styles/
│  ├─ style.css           # Основні стилі
│  └─ responsive.css      # Адаптивні стилі
└─ README.md
```

## Що було стилізовано

### Глобальні стилі
- CSS-змінні для кольорів, відступів та інших параметрів
- Скидання стандартних стилів та box-sizing
- Єдина типографіка для всього сайту
- Кольорова палітра: синій (#2563eb), помаранчевий (#f59e0b), світлий фон (#f8fafc)

### Header та навігація
- Липкий header з тінню
- Горизонтальне меню з Flexbox
- Стани hover та active для посилань
- Плавні переходи при наведенні

### Головна сторінка (index.html)
- Hero-секція з градієнтним фоном
- Стилізовані списки з кастомними маркерами
- Галерея зображень з Flexbox
- Таблиця з чергуванням кольорів рядків та hover-ефектом

### Сторінка "Про мене" (about.html)
- Двоколонкова розкладка з Flexbox
- Оформлені відео та аудіо елементи
- Картки для медіа-контенту
- Підписи до медіа-елементів

### Сторінка контактів (contact.html)
- Повністю стилізована форма
- Стани focus для полів вводу
- Валідація полів (required, email)
- Радіо-кнопки та чекбокси
- Кнопка з hover-ефектом

### Footer
- Темний фон з контрастним текстом
- Центроване вирівнювання

### Адаптивність
- Media queries для екранів ≤768px та ≤480px
- Вертикальна навігація на мобільних
- Одноколонкова розкладка на малих екранах
- Адаптивні розміри шрифтів та відступів
- Горизонтальний скрол для таблиць

## Інструкція запуску

### Варіант 1: Відкриття файлів безпосередньо
1. Завантажте всі файли проєкту
2. Відкрийте `index.html` у браузері

### Варіант 2: Використання Live Server (рекомендовано)
1. Встановіть розширення Live Server у VS Code
2. Відкрийте папку проєкту у VS Code
3. Клацніть правою кнопкою на `index.html`
4. Оберіть "Open with Live Server"

### Варіант 3: Локальний сервер Python
```bash
# Python 3
python -m http.server 8000

# Відкрийте http://localhost:8000 у браузері
```
