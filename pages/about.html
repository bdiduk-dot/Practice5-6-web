Мої навички

Я студент, який вивчає веб-технології та прагне стати професійним розробником. У процесі навчання я освоюю HTML, CSS, JavaScript та інші сучасні інструменти веб-розробки.

Мої основні інтереси включають frontend-розробку, UI/UX дизайн та створення адаптивних веб-інтерфейсів. Я постійно вдосконалюю свої навички через практичні проєкти та вивчення нових технологій.

Технічні навички:

  • HTML5 та семантична розмітка
  • CSS3, Flexbox та Grid
  • Адаптивний веб-дизайн
  • Основи JavaScript
  • Робота з Git та GitHub
